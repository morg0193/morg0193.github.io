About Bootstrap

Open Source

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Mobile First

Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Components Library

Bootstrap comes with an extensive components library that includes buttons and lists to carousels and modals. Components are easy to integrate into any site and require no knowledge of JavaScript to use.

Using Bootstrap

There are two ways to use Bootstrap 4 on your next web project:

1. Download the Compiled Code

You can download Bootstrap 4 on their website: Bootstrap Download

2. Use a CDN

Using a CDN can help shorten website load times by delivering a cached version of the compiled code

CDN Template

To get started with Bootstrap using a CDN, you can use the following starter HTML template:

        
          <!doctype html>
          <html lang="en">
          <head>
          <!-- Required meta tags -->
          <meta charset="utf-8">
          <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

          <!-- Bootstrap CSS -->
          <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/css/bootstrap.min.css" integrity="sha384-Vkoo8x4CGsO3+Hhxv8T/Q5PaXtkKtu6ug5TOeNV6gBiFeWPGFN9MuhOf23Q9Ifjh" crossorigin="anonymous">

          <title>Hello, world!</title>
          </head>
          <body>
          <h1>Hello, world!</h1>

          <!-- Optional JavaScript -->
          <!-- jQuery first, then Popper.js, then Bootstrap JS -->
          <script src="https://code.jquery.com/jquery-3.4.1.slim.min.js" integrity="sha384-J6qa4849blE2+poT4WnyKhv5vZF5SrPo0iEjwBvKU7imGFAV0wwj1yYfoRSJoZ+n" crossorigin="anonymous"></script>
          <script src="https://cdn.jsdelivr.net/npm/popper.js@1.16.0/dist/umd/popper.min.js" integrity="sha384-Q6E9RHvbIyZFJoft+2mJbHaEWldlvI9IOYy5n3zV9zzTtmI3UksdQRVvoxMfooAo" crossorigin="anonymous"></script>
          <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/js/bootstrap.min.js" integrity="sha384-wfSDF2E50Y2D1uUdj0O3uMBJnjuUD4Ih7YwaYd1iqfktj0Uod8GCExl3Og8ifwB6" crossorigin="anonymous"></script>
          </body>

Next Steps

Once you have downloaded Bootstrap or copied the starter template, you are ready to begin building your website using Bootstrap's library of custom classes for layout, content, components, and utilities.

Bootstrap utilizes custom classes to quickly style webpage elements without the need for CSS styling. For example, if you wanted to make your paragraph text blue, you would normally type:

<p style="color:blue;">This text is blue</p>

Whereas with Bootstrap, you could use the following:

<p class="text-primary">This text is blue</p>

To read more about the framework options available with Bootstrap, you can visit the Bootstrap Documentation.

Why use Bootstrap?

Bootstrap is meant to streamline your workflow by allowing you to quickly build high quality web components and layouts by calling pre-built custom classes instead of having to style all your components from scratch. Additionally, it allows you to integrate your own CSS styles to make custom changes to the Bootstrap components. This means you save on time without compromising on creativity.
